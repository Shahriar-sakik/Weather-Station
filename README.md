# Weather-Station
This project is from my university. It is a part of embedded system. where use using ESP32, DHT22, OLED display  and ThingSpeak. Monitors temperature/humidity, shows live data, status, hourly forecast, and uploads sensor readings to the cloud

# Weather Station

Virtual weather station project using ESP32, DHT22, OLED display, and ThingSpeak.

## Features
- Temperature monitoring
- Humidity monitoring
- OLED display interface
- WiFi connectivity
- ThingSpeak cloud logging
- Weather status
- Feels-like condition
- Hourly forecast

## Technologies
- ESP32
- DHT22
- SSD1306 OLED
- Arduino C++
- ThingSpeak
- Wokwi
